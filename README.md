
# Description/Описание
> --------------------------------------------------------------------------------------------------------------------------------
> # **Functional: **
>> 1. Nomenclature: a product directory with the article, name, category, unit of measurement, and minimum balance.
>> 2. Warehouses: accounting by warehouses/storage locations.
>> 3. Document management:*** registration (invoice), write-off (expense), inventory.
>> 4. Real-time balances: recalculating balances during analysis.
>> 5. Order generation: Generates an order to the supplier when the minimum balance is reached.
>> 6. Report Printing: Using the system to generate and print reports.

> # **Функциональность:**
>> 1. Номенклатура: справочник товаров с артикулом, названием, категорией, единицей измерения, минимальным остатком.
>> 2. Склады: учёт по складам/местам хранения.
>> 3. Документооборот: оприходование (приходная накладная), списание (расходная), инвентаризация.
>> 4. Остатки в реальном времени: пересчёт остатков при проведении анализа.
>> 5. Формирование заказов: генерация заказа поставщику при достижении минимального остатка.
>> 6. Печать отчетов: использование системы для генерации и печати отчетов.
